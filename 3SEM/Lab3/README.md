   - Батарейка используется для "включения" фонариков

4. Логика состояний

**6 состояний фонарика:**
1. RectFlashlight - обычный прямоугольный
2. BrokenRectFlashlight - сломанный прямоугольный
3. LitRectFlashlight - горящий прямоугольный (светится)
4. RoundFlashlight - обычный круглый
5. BrokenRoundFlashlight - сломанный круглый
6. LitRoundFlashlight - горящий круглый (светится)

**5 конфликтных объектов:**
1. Stone - камень (ломает фонарик)
2. Screwdriver - отвертка (чинит фонарик)
3. Circle - круг (меняет форму с прямоугольной на круглую)
4. Square - квадрат (меняет форму с круглой на прямоугольную)
5. Battery - батарейка (включает свет)

**Полная логика переходов:**
Прямоугольный + Камень → Сломанный прямоугольный
Сломанный прямоугольный + Отвертка → Прямоугольный
Прямоугольный + Батарейка → Горящий прямоугольный
Горящий прямоугольный (без батарейки) → Прямоугольный
Прямоугольный + Круг → Круглый (смена формы)
Круглый + Камень → Сломанный круглый
Сломанный круглый + Отвертка → Круглый
Круглый + Батарейка → Горящий круглый
Горящий круглый (без батарейки) → Круглый
Круглый + Квадрат → Прямоугольный (смена формы)
