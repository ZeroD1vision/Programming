## Обьяснение

### Как unordered_map находит значение БЕЗ цикла по всем элементам

Когда мы делаем ```пара_кошки["Муся"]```, происходит вот что:

1. Хэш-функция берёт строку ```"Муся"``` и превращает её в число (например, ```1458923476```)
2. Это число используется как индекс в массиве (наша "система комнат")
3. Компьютер сразу переходит по этому индексу в памяти

```cpp
// Псевдокод того, что происходит внутри:
int hash = хэш_функция("Муся"); // → 1458923476  
int index = hash % размер_массива; // → 2 (если массив размером 10)

// ПРЯМОЙ ДОСТУП к ячейке №2 в массиве!
string* результат = массив_значений[2];
```

Никакого цикла! Это как знать точный адрес дома вместо того, чтобы ходить по улице и смотреть на все дома подряд.
